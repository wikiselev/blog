Automated snapshotting with IGV genomic browser

Sat, Oct 20, 2012

Working on my current project I ran into a problem of taking snapshots of protein-RNA binding sites (CLIP data) of proteins from a given protein_list at specific genomic locations (a given gene_list). To take a snapshot of any genomic region I usually use IGV genomic browser (in this post version 2.1.24 (2563) was used). When protein_list (<10 data files) and gene_list (<10 regions) are small snapshotting can be easily done manually.
